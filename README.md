# CLUB_RUN

## Overview

**CLUB_RUN** is a repository that contains project documentation and a database export related to a club-oriented system, most likely developed as part of an academic assignment.

The project includes a written report explaining the concept, requirements, and implementation details, along with a WordPress database dump used for demonstrating or deploying the project website.

---

## Repository Contents

- `Mladen_Andjelkovic_2022_1057.pdf`  
  Project documentation describing the idea, analysis, and results.

- `wordpress.sql`  
  SQL dump of a WordPress database used in the project.

- `.gitignore`  
  Git configuration file for ignoring unnecessary files.

---

1. Open the PDF file to understand the project background and design.
2. Import the database dump into MySQL or MariaDB:
